# GreenWings AI React 7-Minute Deck

Premium 16:9 React pitch deck for GreenWings AI - Aviation Environmental Footprint Advisor.

The redesigned deck uses a dark "Climate Intelligence in Motion" visual system with route arcs, atmospheric bands, dashboard modules, deterministic-tool architecture, data-quality controls, governance, and a decision-brief product view. It is fully code-rendered with an offline-friendly Aptos/Segoe UI font stack and no external image dependencies.

Slides can be opened directly with hash links such as `/#10`.

## Run

```bash
npm install
npm run build
npm run preview
```

Open the local URL printed by Vite. The production build is written to `dist/`.

For active editing, use:

```bash
npm run dev
```

If the dev optimizer hits a local filesystem restriction, use the build and preview commands above.
